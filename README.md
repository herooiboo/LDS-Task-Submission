# Clinic Mini API - EGY2D's Task

## How to install
 - Clone the repo 
 `git clone https://github.com/herooiboo/egy2d.git`
 - run the following commands 
 
  `cp .env.example .env`

  `php artisan key:generate`

    `php artisan migrate`
	
   `php artisan db:seed`

   `php artisan serve`


###### # Enjoy
